## 📖 Sobre o Projeto

**GreenScripts HUB** é um projeto independente que oferece scripts e automações **100% gratuitos** para estudantes do Paraná. Nosso objetivo é contornar restrições desnecessárias e automatizar tarefas repetitivas nas plataformas educacionais estaduais.

```
✅ Gratuito    ✅ Sem cadastro    ✅ Atualizado regularmente
```

## 🔑 Sistema de Chaves

Nossos scripts exigem uma **chave de acesso** para funcionar. A chave é gerada gratuitamente pelo nosso sistema de geração de chaves **[getKey](https://green-scripts.github.io/getKey.html)** — basta completar uma tarefa rápida e a chave é liberada na hora.

> 💡 A monetização por chaves é o que mantém o projeto ativo e os scripts atualizados!

### Como pegar sua chave:

1. Acesse **[green-scripts.github.io/getKey.html](https://green-scripts.github.io/getKey.html)**
2. Complete a tarefa do **LootLabs**
3. Copie a chave gerada e use no script

## 📜 Scripts Disponíveis

### ✍️ Redação Paraná &nbsp;`v2.1.0` &nbsp;![Status](https://img.shields.io/badge/Status-Ativo-brightgreen)

> Automação para a plataforma **Redação Paraná**, usada por estudantes de todo o estado.

| Funcionalidade | Descrição |
|---|---|
| 📋 **Habilitar Colagem** | Desbloqueia o Ctrl+V dentro das redações |
| ✅ **Auto Completar** | Preenche redações automaticamente |
| 🌙 **Dark Mode** | Ativa o modo escuro na plataforma |

#### 🚀 Como instalar e usar

O script funciona como um **bookmarklet** (favorito especial no navegador). Siga os passos:

<details>
<summary><b>💻 Desktop (Chrome / Edge)</b></summary>

**Passo 1 — Acesse a página do script**

Vá até [green-scripts.github.io/scripts.html](https://green-scripts.github.io/scripts.html) e clique em **"Redação Paraná"**.

**Passo 2 — Copie o código**

Clique no botão verde **"Copiar Código"** para copiar o bookmarklet.

**Passo 3 — Abra a barra de favoritos**

Pressione `Ctrl` + `Shift` + `B` para exibir a barra de favoritos abaixo da barra de endereço.

**Passo 4 — Crie o botão de ativação**

1. Clique com o **botão direito** em qualquer espaço vazio da barra de favoritos
2. Selecione **"Adicionar Página"** ou **"Novo Favorito"**
3. No campo **Nome**, escreva: `Redação 1`
4. No campo **URL**, **apague tudo** e **cole** o código copiado (`Ctrl+V`)
5. Clique em **Salvar**

**Passo 5 — Usar**

Entre na plataforma Redação Paraná, abra uma redação e clique em **Redação 1** na sua barra de favoritos. Pronto! 🎉

> ⚠️ O botão **só funciona dentro da página da redação**. Em outros sites não fará nada.

</details>

<details>
<summary><b>📱 Celular Android (Firefox)</b></summary>

> ⚠️ **Requer Firefox** — [Baixe grátis aqui](https://www.mozilla.org/pt-BR/firefox/browsers/mobile/android/)

**Passo 1** — Acesse o site e copie o código do script.

**Passo 2** — No Firefox, abra o site do Redação Paraná, toque nos **⋮ três pontinhos** → **"Adicionar aos favoritos"**.

**Passo 3** — Abra os favoritos (`⋮` → Favoritos), localize o que você salvou, toque nos **⋮** ao lado dele → **"Editar"**.

**Passo 4** — No campo **Nome**, escreva `Redação 1`. No campo **URL**, **apague tudo** e **cole** o código copiado.

**Passo 5** — Toque na seta **←** para salvar. Pronto!

Para usar: entre na redação, toque em `⋮` → **Favoritos** → **Redação 1**. 🚀

</details>

<details>
<summary><b>🖥️ Tablet</b></summary>

O processo é idêntico ao desktop ou celular, dependendo do navegador:

- **Chrome / Edge no tablet:** Siga o tutorial de Desktop acima usando `Ctrl+Shift+B` (se tiver teclado) ou adicione via menu de favoritos do navegador.
- **Firefox no tablet:** Siga o tutorial de Celular acima.

</details>

Veja mais em: [green-scripts.github.io/ScriptCard/RedPR](https://green-scripts.github.io/ScriptsCard/RedPR.html)
