Redação Paraná

Script para redações com colagem automática, contador de palavras e diversas ferramentas para facilitar sua escrita.

Link do Bookmarklet
javascript:(function(){/* link do script aqui */})();

Cole esse link como URL de um favorito no seu navegador.

Como instalar e usar

01

Copie o link do bookmarklet

Clique no botão Copiar acima para copiar o link do script para a sua área de transferência.

02

Exiba a barra de favoritos

No seu navegador, ative a barra de favoritos com o atalho abaixo:

Chrome / Edge
Ctrl+Shift+B
Firefox
Ctrl+Shift+B
Safari (Mac)
⌘+Shift+B
03

Adicione o favorito manualmente

Clique com o botão direito na barra de favoritos e selecione Adicionar página ou Novo favorito.

Nome Redação 1
URL Cole o link copiado aqui
04

Acesse o site e execute o script

Entre na plataforma de redação, clique no favorito Redação 1 na barra de favoritos e o script será carregado automaticamente.

O script só funciona na página correta. Certifique-se de estar na plataforma antes de executar.
